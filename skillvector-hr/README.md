# SkillVector HR

AI-powered resume screening backend for HR recruiters.

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Setup Database:
    - Ensure PostgreSQL is running.
    - Create a database named `skillvector_hr`.
    - Enable `vector` extension: `CREATE EXTENSION vector;`
3.  Run Migrations:
    ```bash
    flask db upgrade
    ```
4.  Run App:
    ```bash
    flask run
    ```
